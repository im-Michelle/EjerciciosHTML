Términos:

Función:
Una función es un bloque de código que realiza una tarea específica y puede ser llamado desde otra parte del código para su ejecución
Let:
En JavaScript, let es una palabra clave que se utiliza para declarar variables con alcance de bloque.
Const:
En JavaScript, const es una palabra clave que se utiliza para declarar constante y tambien para que despues no se pueda cambiar su tipo de dato.
<div>:
Se utiliza para agrupar elementos HTML en un contenedor.
<a>:
Se utiliza para crear un enlace o hipervínculo a otra página web o recurso.
<button>:
Se utiliza para crear un botón interactivo en una página web.
